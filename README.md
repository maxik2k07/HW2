# Система управления рецептами

Консольное приложение для создания рецептов, масштабирования порций и генерации списка покупок.

- `Ingredient` — хранит информацию об одном продукте: название, количество и единицу измерения.
- `Recipe` — объединяет название блюда со списком ингредиентов, позволяет добавлять ингредиенты и масштабировать порции.
- `DietaryRecipe` — расширение рецепта с указанием диетической категории, например "веган" или "без глютена".
- `ShoppingList` — собирает ингредиенты из нескольких рецептов в единый список покупок, суммируя одинаковые продукты.

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/maxik2k07/HW2.git
```
2. Перейдите в папку проекта:
```bash
cd HW2
```
3. Установите зависимости:
```bash
pip install -r requirements.txt
```
